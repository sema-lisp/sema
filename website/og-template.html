Sema Lisp — Dynamic Agentic Programming

A Lisp dialect with first-class LLM primitives, sandboxed tasks, and a reactive execution environment. Implemented in Rust.

Domain sema-lang.com
;; Define an LLM agent with tools
(defagent weather-bot
  {:system "You answer weather questions."
   :model "gemini-2.5-flash"
   :tools [get-weather]})
